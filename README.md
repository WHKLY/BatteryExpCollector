# BatteryExpCollector
This is an Android app for real-time monitoring of battery voltage, temperature, charge level, and exporting data as CSV files.

## Features
- Real-time display of battery voltage, current, and temperature  
- Record historical data and plot charts  
- One-click export of data to CSV file  
- Supports background collection  

## How to Install
1. Clone this repository: `git clone https://github.com/WHKLY/BatteryExpCollector.git`  
2. Open the project with Android Studio  
3. Connect your phone or emulator and click Run  

## Usage Instructions
After opening the app, click the "开始收集" button to begin recording data. Once collection is complete, click "完成收集" to save the CSV file to your phone's storage.
